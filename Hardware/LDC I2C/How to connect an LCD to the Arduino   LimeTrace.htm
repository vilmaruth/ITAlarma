How to connect an LCD to the Arduino

In this tutorial we’ll look at how to connect up a character Liquid Crystal Display (LCD) to an Arduino board. The Arduino IDE comes with an example LCD sketch which uses an Hitachi HD44780 compatible LCD.  We will use a similar LCD. Many LCDs use this controller or a compatible one and you can usually tell by the 16-pin interface. 

Hardware

  • Arduino with USB cable
  • HD44780-compatible LCD - The particular LCD I'm using has QC1602A v2.0 on the back, others have text such as AMC2004A-B-B6NTDW-SP
  • 12 x Jumper wires - You only need to solder 12 of the 16 wires to use the LCD. This will let you to use the 4-bit interface. 
  • 1 x 4K7Ω Potentiometer
  • 1 x 16-pin Male Breakaway Header
  • 10KΩ Resistor
Tools
  • Soldering Iron 
  • Solder
 

LCD 1602 Pin-Out and Interface

On most LCD displays, the LCD pins are numbered from 1-16. If they aren't number please refer to the datasheet that came with the LCD. The table below explains how to connect the LCD pins to the Arduino.  The first thing we need to do is solder the male pin headers to the LCD. Its important that they are soldered to make a decent connection. A simple way to do this is to put the headers in a breadboard and then let the LCD sit on the headers. This leaves both hands free to solder. You only need to connect 12 pins, but it makes sense to solder all 16 in case you need them in future. The LCD pin-outs and the corresponding pin connections on the Arduino are given in Table 1.  

LCD PinSymbolFunctionArduino Pin
1 Vss Ground (0 V) GND
2 Vdd Power (4.5 – 5.5 V) +5V
3 Vo Contrast adjustment Pot. Middle Pin
4 RS H/L register select signal 7
5 R/W H/L read/write signal GND
6 E H/L enable signal 8
11 DB4 H/L data bus for 4-bit mode 9
12 DB5 H/L data bus for 4--bit mode 10
13 DB6 H/L data bus for 4-bit mode 11
14 DB7 H/L data bus for 4-bit mode 12
15 A 'Back Light' Anode - Resistor to 5V +5V
16 K 'Back Light' Cathode GND

Table 1 - LCD to Arduino Connections

  • The 4K7Ω potentiometer is used to adjust the contrast. Connect one side to GND and the opposite side to +5V. Then connect the centre pin to pin 3 on the LCD.  It doesn't matter which side of the pot is at +5V and which is at GND.
  • LCD pins 15,16 are for the “back light” LED. Pin 15 is the anode or positive terminal of the LED and should be connected through a 10KΩ resistor to +5V. 
  • Because we will only be writing, pin 5 will be grounded to show that there will be no reading.

 

Wiring the LCD Display to the Arduino

Follow the above table and circuit diagram in and wire up the LCD using the wires provided. The wires are colour coded to help you.

 Wire Colour Description
Red +5V (Vcc)
Black Ground
White Data Lines
Blue Control Lines

Sketch Code

To run the sketch code you need to:

  • Open up the Arduino IDE 
  • Select File -> Examples -> LiquidCrystal -> HelloWorld  
  • Open the sketch and upload it to your Arduino board  
  • Once you are comfortable with the code you can change the text, display variables, sensor values or even use analogRead().
The Arduino LiquidCrystal library allows an Arduino board to control LCDs that use the commonly employed Hitachi HD44780 or compatible controller. The library can use either 4 or 8 data lines in addition to the rs, enable and optional rw control lines.  In this example we use Arduino pins 12,11,10, and 9 to connect to LCD pins 11,12,13 and 14, respectively (see Table 1).  
/*
LiquidCrystal Library - Hello World

Demonstrates the use a 16x2 LCD display. The LiquidCrystal
library works with all LCD displays that are compatible with the
Hitachi HD44780 driver. There are many of them out there, and you
can usually tell them by the 16-pin interface.

This sketch prints "Hello LimeTrace!" to the LCD
and shows the time.

The circuit:
* LCD RS pin to digital pin 7
* LCD Enable pin to digital pin 8
* LCD D4 pin to digital pin 9
* LCD D5 pin to digital pin 10
* LCD D6 pin to digital pin 11
* LCD D7 pin to digital pin 12
* LCD R/W pin to ground
* 10K resistor:
* ends to +5V and ground
* wiper to LCD VO pin (pin 3)

Library originally added 18 Apr 2008
by David A. Mellis
library modified 5 Jul 2009
by Limor Fried (http://www.ladyada.net)
example added 9 Jul 2009
by Tom Igoe
modified 22 Nov 2010
by Tom Igoe
This example code is in the public domain.

http://www.arduino.cc/en/Tutorial/LiquidCrystal
*/

// include the library code:
#include <LiquidCrystal.h>

// Initialize the library with the interface pins
LiquidCrystal lcd(7, 8, 9, 10, 11, 12);

void setup() {
// set up the LCD's number of columns and rows 
lcd.begin(16, 2);
// Print a message to the LCD
lcd.print("hello LimeTrace!");
}

void loop() {
// set the cursor to column 0, line 1
// (line 1 is the second row, since counting begins with 0)
lcd.setCursor(0, 1);
// print the number of seconds since reset:
lcd.print(millis()/1000);
}
