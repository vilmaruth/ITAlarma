Tutorial: Arduino I/O Port Expander with PCF8574 Tutorial: Arduino I / O Port Expander con PCF8574

  • Posted by GarageLab on May 1, 2012 at 10:55am Publicado por Garagelab el 1 de mayo de 2012 a las 10:55 am
  • View Blog Ver Blog

The PCF8574 is a 8 bits I/O port expander that uses the I2C protocol. El PCF8574 está a 8 bits de expansión del puerto de E / S que utiliza el protocolo I2C. That protocol communicates using a 2-wire serial interface, where one wire is a serial clock (SCL) and the other is for serial data (SDA). Ese protocolo de comunica utilizando una interfaz en serie de 2 hilos, en donde uno de los cables es un reloj serie (SCL) y la otra es para datos en serie (SDA). With this IC, you can use only 2 ports of your Arduino board to control up to 8 digital I/O ports. Con este circuito, se puede utilizar sólo 2 puertos de la placa Arduino para controlar hasta 8 puertos de E / S digitales. In the I2C protocol, each IC has a different address, in the case of PCF8574, you have a 3 bits address, which enable you to connect up to 8 devices, having 64 ports in total. En el protocolo I2C, cada IC tiene una dirección diferente, en el caso de PCF8574, tienes un bits de dirección 3, que le permiten conectar hasta 8 dispositivos, con 64 puertos en total.

We can take a look on the PCF8574 pinout in the data datasheet : Podemos echar un vistazo en el PCF8574 pinout en los datos de hoja de datos :

Where: A0,A1,A2 are address pins Dónde: A0, A1, A2 son pines de dirección

P0,P1,P2,P3,P4,P5,P6,P7 are digital I/O ports (each one is 1 bit) P0, P1, P2, P3, P4, P5, P6, P7 son puertos de E / S digitales (cada uno es de 1 bit)

SDA,SCL are the I2C pins to communication with the microcontroller. SDA, SCL son los pines I2C para la comunicación con el microcontrolador.

To make the IC work, it is necessary to set and address, again we can find this information on the datasheet: Para hacer el trabajo de IC, es necesario establecer y abordar, de nuevo podemos encontrar esta información en la hoja informativa:

where: L means LOW or 0V(GND) donde: L significa BAJO o 0V (GND)

H means HIGH or VCC H significa ALTO o VCC

To define an address set A0, A1 and A2 to H or L as you desire. Para definir un conjunto de direcciones A0, A1 y A2 de H o L como usted desea. Remember that each IC should has a different address. Recuerde que cada IC debe tiene una dirección diferente.

Checking the data interface, we can see how the I2C interface will work on the PCF8574: Comprobación de la conexión de datos, podemos ver cómo la interfaz I2C trabajará en la PCF8574:

If you send a I/O data byte, it will turn on or off the corresponding pins P1 to P7 according to the byte send. Si se envía un byte de datos de E / S, se encenderá o apagará los pines correspondientes P1 a P7 según el byte enviar. For example, if you send the number 32, it will be 0010000 in binary, which will turn on only the P5 output. Por ejemplo, si envía el número 32, será 0010000 en binario, que se convertirá en la única salida de la P5. For reading, the same is true, if you receive the number 32, means that only the port P5 was on at that moment. Para la lectura, lo mismo es cierto, si usted recibe el número 32, que significa que sólo el P5 puerto estaba apagado en ese momento.
To test the IC, let's wire an Arduino board,2 PCF8574, some LEDs and buttons: Para probar la IC, vamos alambre una placa Arduino, 2 PCF8574, algunos LEDs y botones:
The left PCF8574 is a receiver, and it will control the LEDs. El PCF8574 izquierda es un receptor, y será el control de los LEDs. The right one will be a emitter and will transmit the button's status. La de la derecha será un emisor y transmitirá el estado del botón. When you press a button, the Arduino will read it from the I2C interface and will command the left IC to turn the corresponding LED on. Al presionar un botón, el Arduino leerá desde la interfaz I2C y mandará a la IC izquierda para girar el correspondiente LED en.
